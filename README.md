# Mutual Fund Performance Analytics Dashboard

## Project Overview
An end-to-end financial analytics project analyzing 
20 mutual funds across Equity, Debt, and Hybrid 
categories using Excel modelling and Power BI 
visualization.

## Tools Used
- Microsoft Excel (Financial Modelling)
- Power BI Desktop (Dashboard)
- Power Query (Data Transformation)
- DAX (Measures & Calculations)

## Key Metrics Calculated
- NAV Trends (Q1 2020 to Q1 2025)
- 1Y / 3Y / 5Y CAGR
- Sharpe Ratio
- Jensen's Alpha
- Beta
- Expense Ratio Impact

## Key Findings
- Equity funds averaged 17% CAGR vs 8% for Debt
- Higher expense ratios negatively correlated 
  with net returns
- Small-cap funds showed 3x returns but 6x 
  volatility vs debt funds
